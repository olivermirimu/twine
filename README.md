# twine

A command line application hat allows a user to make posts and carry out other twitter related account functionality using a command line interface.

# Getting started

1. Clone the repository
2. Run npm install in the project root directory.
3. Run twine confugure consumer. You will be promted for your twitter developer api key and secret. (These shall be stored in your repective OS's credential manager).
4. Run twine configure account. You will be promted to open a browser where you will login into your twitter account.
